# ChatGPT Software Question

Raycast extension to launch ChatGPT and ask about the current frontmost application.

## Features

- Detects the frontmost macOS application
- Launches ChatGPT and pastes a prompt asking about that app

## Commands

| Command | Description |
|---------|-------------|
| ChatGPT Software Question | Start a ChatGPT conversation about the frontmost app |

## Install

```bash
bun install
bun run build
bun run dev
```
